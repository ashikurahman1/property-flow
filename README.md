# Property Flow - Real Estate MERN App

A full-stack **Real Estate Web Application** built with the **MERN stack**
(MongoDB, Express, React, Node.js).  
Users can **search properties**, **create**, **update**, and **delete
listings**, and manage their accounts with secure authentication.

---

## Features

### Property Management

- Search properties by name, location, price, etc.
- View detailed information for each listing.
- Responsive and clean UI.

### User Actions

- Create new property listings
- Edit existing listings
- Delete listings
- Upload property images

### Authentication

- JWT-based login & registration
- Role-based permissions (optional)
- Secure API routes

---

## Tech Stack

### **Frontend**

- React.js
- React Router
- Axios / Fetch API
- Tailwind CSS (or your chosen styling framework)

### **Backend**

- Node.js
- Express.js
- MongoDB + Mongoose
- JSON Web Tokens (JWT)
